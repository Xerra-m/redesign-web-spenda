# 🚀 SMP N 2 KOTA TEGAL - Web Redesign

An independent web development project designed to sharpen frontend skills and experiment with modern UI/UX design principles. The primary focus is on implementing a responsive layout using Tailwind CSS and a component-based architecture with React.

## 🛠️ Tech Stack
* **Framework**: React.js with Vite
* **Styling**: Tailwind CSS
* **Deployment**: GitHub Pages
* **Editor**: Acode (Mobile Development)

## 📌 Why This Project?
What started as a spontaneous "boredom" project has evolved into a serious practice in building a clean, performant, and mobile-friendly web interface.

## 💡 Key Features
* **Responsive Design**: Fluid layout optimized for mobile browsing.
* **Component-Based**: Modular code structure to facilitate future scalability.
